
# yutodev

Personal Blog

## Installation

Install my-project with pnpm

```bash
  git clone https://github.com/code-yy/yutodev.git

  cd yutodev

  pnpm install

  pnpm dev
```
    

## Tech Stack

**Client:** React, Next.js, TypeScript, Mantine, TailwindCSS, pathpida, Prettier, ESLint

**Server:** microCMS
